My UI and UX

2022/08

I'm pretty happy with my UI and UX for this website, because I think it remains consistent and easy to navigate. I struggled to line up my nav bar with my logo but that was a technical issue that I can fix in my next prototype.

I think the hamburger menu is a good part of the UI, as it's a well established symbol for the menu on mobile versions of websites. For this prototype I really focused on getting the responsive design working and I'm very proud of myself for managing to make it all work.

On my blog page,in terms of the responsive design, I think there's probably a better way to show the other blog posts than at the bottom of the page, but overall I'm fairly happy with how the UI turned out on every page.
